# 🏆 СпелеоСтарт: Инструкция судьи

## 🔹 Быстрый старт

1. Откройте приложение → **Соревнования** → **➕ Новое**
2. Введите название, дату, место → **СОЗДАТЬ**
3. Добавьте участников через **Персоны** → **➕ Новая персона**
4. **ВАЖНО:** Нажмите **СТАРТ** → **НАЧАТЬ СОРЕВНОВАНИЯ** для запуска таймера

## 🔹 Отображение времени

- Время отображается **относительно начала соревнования**
- Старт: `+00:00:00` (первая команда), далее с интервалом
- Финиш: относительное время от старта (например, `+01:45:30`)
- Если время не отображается — нажмите **НАЧАТЬ СОРЕВНОВАНИЯ**

## 🔹 Корректировка времени финиша

1. Откройте **Команды** → выберите команду
2. Нажмите на время финиша (синий текст)
3. Введите новое время в формате ЧЧ:ММ:СС
4. Нажмите **СОХРАНИТЬ**

## 🔹 Путевой лист (ПЛ)
### Быстрый доступ к заполнению

1. На главном экране нажмите на плитку **«Путевые»** (рядом с счётчиком ожидающих команд)
2. В появившемся диалоге выберите команду из списка:
   - Слева отображается номер команды в квадрате
   - Справа — класс и первые две фамилии участников
3. Вы перейдёте в карточку команды, где **сразу откроется режим редактирования КП**
4. Отмечайте взятые КП нажатием на цветные кружки:
   - 🟢 Зелёный — КП взят
   - 🟠 Оранжевый — технический КП (требует ввода отсечки)
   - ⚪ Серый — не взят
5. Для технических КП: долгий тап по кружку → введите отсечку и штраф
6. После заполнения:
   - Нажмите **«📝 Подписать секретарём»** (пароль `secret123`)
   - Нажмите **«⚖️ Подписать судьёй»** (пароль `judge123` или `1234`)

### Статусы ПЛ в карточке команды:
- ✅ **ПУТЕВОЙ ЛИСТ** (зелёный) — подтверждён, данные закрыты
- 📝 **ПУТЕВОЙ ЛИСТ** (оранжевый) — заполняется частично
- 📋 **ПУТЕВОЙ ЛИСТ** (серый) — не заполнен
- ❌ **ПУТЕВОЙ ЛИСТ** (красный) — отменён (сняты/LOST)


При заполнении технического КП:

1. **Отсечка** — вводится в формате `ММ:СС` (минуты:секунды)
    - Пример: `01:30` = 1 минута 30 секунд
    - Пустое поле = отсечки нет
    - **Важно:** Поле с ошибкой будет подсвечено красным, кнопка "СОХРАНИТЬ" станет неактивной

2. **Штраф** — целое число
    - Положительное = бонус (например, `5`)
    - Отрицательное = штраф (например, `-3`)
    - Пустое поле = 0

3. **Валидация:**
    - Нельзя сохранить некорректные данные
    - Ошибка отображается под полем красным текстом
    - Примеры ошибок:
        - `"12"` → нужно `12:00` или `00:12`
        - `"01:99"` → секунды не могут быть 99
        - `"abc"` → только цифры

### Заполнение ПЛ:
1. Нажмите **📄 ЗАПОЛНИТЬ / ПРОДОЛЖИТЬ**
2. Отметьте взятые КП и ошибки
3. Для технических КП: укажите отсечку и штраф
4. **Подтверждение:**
   - Секретарь: пароль `secret123`
   - Судья: пароль `judge123` или `1234`

### Мастер-правка (для исправления ошибок):
1. Нажмите **🔓 Мастер-правка** в карточке команды
2. Введите пароль `devdebug`
3. Внесите изменения в путевой лист
4. Нажмите **💾 СОХРАНИТЬ**
5. **ВАЖНО:** Изменения сохраняются навсегда, даже после перезапуска приложения
6. Статусы "подписано" (секретарь/судья) НЕ сбрасываются

## 🔹 Визуализация КП

- КП отображаются цветными кружками в карточке команды:
   - 🟢 Зелёный — КП взят
   - 🟠 Оранжевый — взят с ошибкой
   - ⚪ Серый — не взят
