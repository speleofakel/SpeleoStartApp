# CHANGELOG.md · SpeleoStart

## [Невыпущенное] — 30.04.2026

### ✨ Добавлено
- `PersonDetailScreen`: экран карточки персоны с режимами просмотра/редактирования
- `TitleCaseTextField`: компонент с автозаглавной первой буквы и переходом по пробелу
- Маска даты рождения `ДД.ММ.ГГГГ` с автоматической расстановкой точек
- Цветная метка возраста (зелёный 18+, жёлтый 14-17, оранжевый <14 и >70)
- Чёрный список: добавление/удаление через `PersonDetailVM.blacklistPerson()`
- `PhoneFormatter`: маска телефона `+7 (XXX) XXX-XX-XX`

### 🔄 Изменено
- `DateValidator` переписан на `java.util.Calendar` (без `java.time` для совместимости с API 24)

### 🐛 Известные проблемы (НЕ РЕШЕНЫ)
- **Валидация даты:** поле не подсвечивается красным при незавершённом вводе (менее 10 символов)
- **Валидация даты:** `isError = true` только при длине ровно 10 символов
- **Автопереход:** не срабатывает при достижении 10 символов, если дата некорректна

### 🚫 NEVER-DO
- Не использовать `java.time.LocalDate` на API < 26 (краш `NoClassDefFoundError`)
- Не повышать версию БД без изменения полей `@Entity`
- Не давать инструкции частями — только полные файлы
