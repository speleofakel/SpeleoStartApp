## [4.5] — 11.05.2026

### Добавлено
- **Автоматическое открытие режима редактирования КП** при переходе в карточку finished команды с незаполненным путевым листом

  - Отображение первых двух фамилий участников
  - Переход сразу в карточку команды вместо старого экрана ПЛ

### Улучшено
- **Главный экран без активного соревнования** теперь показывает только 3 плитки: Соревнования, Персоны, Настройки
- **Плитки меню** получили тень (elevation 4.dp) для лучшей визуализации
- **Расстояние между плитками** уменьшено с 12dp до 6dp
- **TopAppBar** уменьшен до 52dp с минимальными отступами

### Исправлено
- Навигация из диалога путевых листов теперь ведёт в `TeamCardScreen` вместо устаревшего `RouteCardDetail`
 
## [4.4] — 07.05.2026

### Исправлено (КРИТИЧЕСКОЕ)
- **Мастер-правка путевого листа теперь реально сохраняет изменения в БД**
    - Проблема: изменения мастера отображались только до выхода из карточки
    - Причина: `saveMasterChanges()` не обновлял существующие записи корректно
    - Решение: переписан метод с прямой работой через `TeamRouteCardRepository`
    - Добавлена верификация сохранения через повторное чтение из БД

### Улучшено
- Логирование процесса мастер-сохранения для отладки
- Принудительная перезагрузка ПЛ после мастер-сохранения
