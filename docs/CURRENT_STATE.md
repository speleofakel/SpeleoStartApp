## 3. СЦЕНАРИИ ДЛЯ ПРОВЕРКИ

### Замена участника
1. Открыть карточку команды
2. Кликнуть по участнику → кнопка "Заменить"
3. В поиске ввести фамилию существующей персоны → выбрать → замена работает
4. В поиске ввести новую фамилию → нажать "➕ Создать: фамилия"
5. **ОЖИДАЕТСЯ:** персона создаётся с нормализацией Title Case
6. **ФАКТ:** персона сохраняется с нормализацией ✅

### Остановка главного таймера (НОВОЕ)
1. Открыть экран СТАРТ
2. Нажать "НАЧАТЬ СОРЕВНОВАНИЯ"
3. Долгий тап по главному таймеру
4. Ввести пароль 1234 или devdebug
5. **ОЖИДАЕТСЯ:** таймер останавливается и сбрасывается
6. **ФАКТ:** реализовано, требует тестирования

### Корректировка времени финиша (НОВОЕ)
1. Открыть экран КОМАНДЫ
2. Найти finished команду
3. Долгий тап по карточке команды
4. Ввести новое время в формате ЧЧ:ММ:СС
5. **ОЖИДАЕТСЯ:** время финиша обновляется
6. **ФАКТ:** реализовано, требует тестирования

### Старт команды
1. Открыть экран СТАРТ
2. Нажать "НАЧАТЬ СОРЕВНОВАНИЯ"
3. Выбрать следующую команду → нажать "▶ СТАРТ"
4. **ОЖИДАЕТСЕТСЯ:** команда переходит в статус started, таймер сбрасывается
5. **ФАКТ:** реализовано, требует тестирования на устройстве
