## 11.05.2026 — Валидация времени отсечки

| Правка | Причина | Решение |
|--------|---------|---------|
| Тихое отбрасывание некорректной отсечки | Пользователь не видит ошибку | Кнопка "СОХРАНИТЬ" блокируется, красная ошибка под полем |
| Placeholder "01:30" выглядит как введённое | Сбивает с толку | Placeholder "мм:сс" серым полупрозрачным шрифтом |
| Нет подсказки формата | Пользователь гадает | Supporting text с примером "01:30" |
| Обычный шрифт | Неудобно для времени | Моноширинный шрифт |
| Штраф "abc" тихо = 0 | Потеря данных | Валидация целого числа, блокировка кнопки |


## 06.05.2026 — Карточка команды: кластеры КП, относительное время, мастер-правка

| Правка | Причина | Решение |
|--------|---------|---------|
| КП в виде кластеров | Список КП не наглядный | Цветные кружки (зелёный/оранжевый/серый) |
| Не помещаются на экран | 12 кружков в ряд | Уменьшен размер (36dp), перенос по 6 элементов |
| Время — прочерки | Нет старта соревнования | Отображать "—:—:—" до нажатия "НАЧАТЬ СОРЕВНОВАНИЯ" |
| Время астрономическое | Неудобно для судейства | Относительное время от старта соревнования |
| Нет корректировки финиша | Ошибки в фиксации | Кликабельное время финиша → диалог ввода |
| После мастер-правки нет "СОХРАНИТЬ" | UX проблема | Добавлена отдельная кнопка сохранения |
| Требуются пароли после мастер-правки | Логическая ошибка | Мастер-сохранение сохраняет статусы "подписано" |
| Статус "ПРОДОЛЖИТЬ" после правки | Неверный статус | После сохранения → CONFIRMED |

## 05.05.2026 — Остановка таймера и правка финиша

| Правка | Причина | Решение |
|--------|---------|---------|
| Остановка главного таймера | Экстренная остановка при сбое | Долгий тап → пароль `1234`/`devdebug` |
| Индивидуальная корректировка финиша | Групповой финиш не всегда точен | Долгий тап по карточке finished команды |
| Счётчик `0/12` на плитке "Путевые" | Визуализация прогресса | `pending/total` с цветом (серый/оранжевый/зелёный) |
| ПЛ в карточке команды | Удобство судьи | Статистика + список КП с цветами |
| Мастер-правка ПЛ | Исправление ошибок без переподтверждения | Пароль `devdebug`, разблокировка 1 листа |

## 03.05.2026 — Кнопки "+" и "✖" в поиске

| Правка | Причина | Решение |
|--------|---------|---------|
| Кнопка "+" в поле поиска | Нельзя создать персону при наличии однофамильцев | Зелёная кнопка, создание с предзаполненной фамилией |
| Кнопка "✖" в поле поиска | Быстрая очистка | Красная кнопка, появляется только при непустом поле |

## 02.05.2026 — Title Case для русского языка

| Правка | Причина | Решение |
|--------|---------|---------|
| Замена `Locale.getDefault()` | На устройствах с английской локалью русские буквы не обрабатывались | `Locale("ru", "RU")` |
